Hello, my name is RichText.

As you can see, I'm another rich-text editor. So why am I different?

  • I'm made of web-components

  • You don't have to initialize me, I just work

  • Plugins that work for me, are standalone, they also don't need to be initialized.

  • You will use me the way, the real web rich text editor should be used - using plain HTML

Something other interesting things about me:

  • If you want to set HTML content, you do it in rich-text-content custom-element

  • If you want to set content from JSON, to do id by passing stringified value to rich-text (me!) custom-element

  • I can make @John Doe mentions and #markwallace hashtags!

    • It's not SUPER easy to make a list of options because I'm a web-component, but you can do it by placing a getItems(query) method on my Mention and Hashtag plugins when importing them.

    • Check out the main file from which I export everything, to see how it works!

  • I can also

    • make some

      • nested lists

    • as well!
